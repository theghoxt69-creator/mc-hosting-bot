# MC Hosting Bot

## Setup Instructions
1. **Clone the Repository**:  
   Run the following command in your terminal:
   ```bash
   git clone https://github.com/<owner>/mc-hosting-bot.git
   cd mc-hosting-bot
   ```

2. **Install Dependencies**:  
   Use a package manager like npm or yarn:
   ```bash
   npm install
   # or
   yarn install
   ```

3. **Configuration**:  
   Copy the `.env.example` file to `.env` and fill in your configuration settings.
   ```bash
   cp .env.example .env
   ```

4. **Start the Bot**:  
   After setting up, run:
   ```bash
   npm start
   ```

## Deployment Guide
1. **Build the Project**:  
   Run the following command to build the project:
   ```bash
   npm run build
   ```

2. **Deploy**:  
   Upload your files to the production server and start the bot with:
   ```bash
   npm start
   ```

3. **Monitor logs**: Check logs to ensure everything is running smoothly.
